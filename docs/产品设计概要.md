# Yours·凝刻 — 产品设计概要 v1.0

> 状态：✅ 已确认 | 确认日期：2026-03-12 | 确认人：产品负责人 + 主策

---

## 1. 产品基本信息

| 项目 | 内容 |
|------|------|
| **产品名称** | Yours·凝刻 |
| **产品形态** | 微信小程序 |
| **产品定位** | AI 驱动的个性化 Q 版 3D 手办定制平台 |
| **目标用户** | 18-35 岁年轻用户，潮玩/盲盒爱好者，有个性化送礼需求的消费者 |
| **竞品参考** | 泡泡玛特盲盒手办风格（Q 版、大头、彩色涂装、精致） |
| **核心价值** | "把你变成一个手办"——照片到实体，全流程数字化定制 |

## 2. 设计原则

1. **仪式感优先** — 开箱体验是产品核心卖点之一
2. **流程极简** — 上传照片 → 预览模型 → 定制下单，三步完成
3. **模块化定价** — 基础手办 + 可选周边，灵活组合
4. **社交传播** — 分享功能驱动裂变增长

## 3. 用户核心旅程

```
注册登录 → 上传全身照 → AI生成3D模型（等待/通知）→ 360°预览模型
   → 确认模型 → 进入产品定制（选尺寸/选周边/选包装盒）
   → 价格预览 → 确认订单 → 微信支付 → 订单跟踪 → 完成
```

## 4. 功能模块定义

### 4.1 用户体系
| 功能 | 说明 |
|------|------|
| 微信授权登录 | 小程序标准流程，获取头像昵称 |
| 手机号绑定 | 用于收货和订单通知 |
| 收货地址管理 | 增/删/改/默认地址 |
| 个人信息编辑 | 头像、昵称 |

### 4.2 照片上传与管理
| 功能 | 说明 |
|------|------|
| 上传全身照 | 相册选取 / 现场拍照 |
| 拍照引导 | 全身、正面、光线、姿势等要求提示 |
| 照片裁剪 | 辅助用户调整照片 |
| 历史照片 | 查看/删除已上传的照片 |

### 4.3 AI 3D 模型生成
| 功能 | 说明 |
|------|------|
| AI 生成引擎 | 全身照 → Q 版彩色 3D 手办模型 |
| 生成状态 | 排队中 → 生成中 → 完成/失败 |
| 页面内等待 | 等待动画 + 预计时间 |
| 消息通知 | 离开小程序后，模板消息/订阅消息通知用户回来 |
| 失败处理 | 提示原因 + 重试/换照片 |
| 多模型支持 | 一个用户可生成多个模型（不同照片），各自独立下单 |

### 4.4 3D 模型在线预览
| 功能 | 说明 |
|------|------|
| 360° 旋转预览 | 手势旋转/缩放查看模型 |
| 光影展示 | 真实光影效果 |
| 操作引导 | 首次查看时的手势引导 |
| 确认/重新生成 | 满意 → 进入定制；不满意 → 重新生成 |

### 4.5 产品定制
| 定制项 | 类型 | 说明 |
|--------|------|------|
| **手办模型** | ✅ 必选 | 按尺寸定价（如 S/M/L），材质暂时固定单一材质 |
| **模型照片打印** | 可选 | AI 生成的模型渲染图打印，多种打印类型可选（不同载体不同价格） |
| **真人原照片打印** | 可选 | 用户上传的原照打印，多种打印类型可选（不同载体不同价格） |
| **包装盒** | 可选 | 同材质手办级包装盒，正方体，8 种基础颜色可选 |
| **套餐组合** | 可选 | 预设的优惠组合套餐 |

#### 打印类型（模型照片 & 真人照片共用）
| 打印类型 | 说明 |
|---------|------|
| 相纸打印 | 标准照片纸 |
| 卡片打印 | 精致小卡 |
| 亚克力板打印 | 高端展示 |
| 其他（可扩展） | 后续增加 |

> 每种打印类型独立定价，由后台管理

#### 包装盒设计要点
- **材质**: 与手办同材质，本身也是展示品
- **结构**: 一体式六面正方体，可展开取出手办，可合拢后将手办摆放在顶部
- **颜色**: 8 种基础色可选
- **表面**: 暂时无印刷内容
- **扩展预留**: 后续加小卡扣用于放置卡片/亚克力板

#### 套餐示例（后续可调整）
| 套餐 | 包含内容 |
|------|---------|
| 基础款 | 手办(S) |
| 精致款 | 手办(M) + 包装盒 + 模型卡片 |
| 豪华款 | 手办(L) + 包装盒 + 双面照片打印 + 亚克力板 |

### 4.6 价格计算与订单支付
| 功能 | 说明 |
|------|------|
| 实时价格计算 | 根据定制选项动态计算总价 |
| 价格预览页 | 明细展示每项费用 |
| 订单确认页 | 商品清单 + 定制详情 + 收货地址 + 总价 |
| 微信支付 | 小程序内支付 |
| 订单状态 | 待支付 → 已支付/制作中 → 已发货 → 已完成 |
| 订单详情 | 查看定制内容、物流信息 |

### 4.7 "我的"个人中心
| 功能 | 说明 |
|------|------|
| 个人信息 | 头像、昵称、手机号 |
| 我的订单 | 全部/待支付/制作中/已发货/已完成，支持筛选 |
| 我的模型 | 历史生成的所有 3D 模型列表，可查看/可重新下单 |
| 收货地址 | 地址管理 |
| 分享功能 | 分享小程序/分享模型给好友 |
| 联系客服 | 在线客服入口 |
| 关于我们 | 品牌介绍 |
| 用户协议/隐私政策 | 合规必备 |

## 5. 模块间接口定义

| 接口节点 | 上游模块（输出） | 下游模块（输入） | 传递数据 |
|---------|---------------|---------------|---------|
| 确认模型 → 进入定制 | AI 生成 & 预览（策划-A） | 产品定制（策划-B） | 模型 ID、模型预览图、原照片 |
| 定制完成 → 提交订单 | 产品定制（策划-B） | 订单支付（策划-B） | 商品清单、定制选项、总价 |
| 订单列表入口 | 个人中心（策划-A） | 订单管理（策划-B） | 订单 ID、订单状态 |
| 我的模型 → 重新下单 | 个人中心（策划-A） | 产品定制（策划-B） | 模型 ID |

## 6. 用户讨论纪要

### 2026-03-12 首次讨论
- 确认产品定位为 AI 个性化 Q 版 3D 手办定制
- 打印类型可自定义，不同载体不同价格
- 包装盒与手办同材质，本身是展示品，手办摆放在包装盒顶部
- 包装盒 8 种基础颜色，暂无印刷内容，预留小卡扣扩展
- 手办按尺寸定价，材质暂时固定，支持套餐组合
- AI 生成等待支持页面内等待 + 离开后消息通知两种方式
- 支持多模型生成，各自独立下单
- 支持分享功能

### 2026-03-12 待确认项决策
- **AI 生成耗时**：暂不设时间要求，先验证功能和 AI 可落地性
- **免费次数**：每人每日 2 次免费生成
- **3D 模型格式**：交由技术 Agent 评估决定
- **定价**：MVP 阶段全部使用 ¥0.01 测试价，先跑通全流程
- **制作周期**：暂无要求，先跑模型。需支持模型文件导出供第三方制作验收
- **退款政策**：已拟通用版（支付后 24h 内可全额退，制作中不可退，质量问题 7 天可退）
- **支付测试**：不进行真实付费测试

## 7. MVP 阶段策略

> 当前处于 **MVP 验证阶段**，核心目标是验证 AI 生成 3D 模型的可落地性和全流程可跑通。

| 维度 | MVP 策略 | 正式版计划 |
|------|---------|-----------|
| AI 生成 | 不限耗时，先验证效果 | 优化速度，展示预计时间 |
| 定价 | 全部 ¥0.01 测试价 | 商务定真实价 |
| 支付 | 不真实付费 | 接入微信支付 |
| 制作 | 导出模型文件，第三方手工验收 | 对接生产线自动化 |
| 免费次数 | 每日 2 次 | 视成本调整 |
| 退款 | 通用政策 | 法务审核完善 |
