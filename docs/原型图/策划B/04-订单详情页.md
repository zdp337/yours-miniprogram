## 交互说明

| 序号 | 元素 | 交互行为 | 跳转/效果 |
|------|------|---------|----------|
| 1 | "复制"（订单号）| 点击 | 复制到剪贴板 + toast提示 |
| 2 | "复制"（快递号）| 点击 | 复制到剪贴板 + toast提示 |
| 3 | "立即支付" | 点击 | 调起微信支付 |
| 4 | "取消订单" | 点击 → 弹窗确认 | 取消订单 |
| 5 | "确认收货" | 点击 → 弹窗确认 | 订单状态→已完成 |
| 6 | "联系客服" | 点击 | 调起微信客服 |
| 7 | "申请退款" | 点击 | → 退款申请页 |
| 8 | "再来一单" | 点击 | → 产品定制页（携带modelId） |
| 9 | "重新下单" | 点击 | → 产品定制页（携带modelId） |
