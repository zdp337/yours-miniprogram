## 依赖关系

```
PM(项目初始化) ──→ 研发-A(后端API) ──→ 研发-B(前端对接)
      │
      └──→ 研发-C(前端页面UI，可并行)
```

- 研发-C 不依赖后端，Day 1 即可开始做静态 UI
- 研发-B 依赖研发-A 的 API，Day 1-2 做静态 UI，Day 3 开始对接
- 研发-A Day 1 建表，Day 2 开始写 API

## 验收标准

1. 打开小程序 → 显示欢迎页 → 点击登录 → 微信授权 → 进入首页
2. 首页有 Tab 栏，可切换首页/定制(空)/我的
3. 个人中心显示用户头像昵称，手机号绑定入口
4. 可编辑个人信息（头像、昵称）
5. 可新增/编辑/删除/设默认收货地址
6. 所有 API 返回统一格式：`{ code: 0, data: {}, message: "" }`
