- **模块名称**: 产品定制
- **负责人**: 策划-B
- **版本**: v1.0
- **最后更新**: 2026-03-12
- **状态**: 初稿

---

## 功能概述

产品定制是用户将 AI 生成的虚拟模型转化为实体商品的核心环节。用户在此页面选择手办尺寸（必选），以及可选的照片打印、包装盒等周边产品，系统实时计算价格，最终生成订单。

**入口数据**（来自策划-A 的 3D 预览模块）:
- `modelId` — 确认使用的模型 ID
- `modelPreviewUrl` — 模型预览图 URL
- `originalPhotoUrl` — 用户上传的原始照片 URL

## 详细设计

### 功能点 1：手办尺寸选择（必选）

- **描述**: 用户选择手办的实体尺寸，按尺寸定价
- **用户操作流程**:
  1. 进入定制页，手办尺寸为第一个选择项
  2. 展示 3 种尺寸的对比卡片
  3. 默认选中 M 码（推荐款，标记"推荐"标签）
  4. 点击切换尺寸，右侧模型预览图同步更新尺寸比例参考
- **尺寸规格**:

| 尺寸 | 高度 | 价格 | 说明 |
|------|------|------|------|
| S（迷你款） | 约 8cm | ¥待定 | 精致小巧，适合桌面摆放 |
| M（标准款） | 约 12cm | ¥待定 | ⭐ 推荐，最佳观感 |
| L（豪华款） | 约 18cm | ¥待定 | 大尺寸，细节丰富 |

- **逻辑规则**:
  - 必选项，用户必须选择一个尺寸才能继续
  - 默认选中 M 码
  - 材质固定（v1 不可选），展示材质说明文字
  - 切换尺寸时价格实时更新
- **异常处理**:
  - 某尺寸暂时缺货：展示"暂时缺货"标签，不可选择

### 功能点 2：模型照片打印（可选）

- **描述**: 将 AI 生成的模型渲染图打印为实体，多种打印类型可选
- **用户操作流程**:
  1. 在"可选周边"区域，展示"模型照片打印"卡片
  2. 开关式选择：不需要 / 需要
  3. 选择"需要"后展开打印类型选择
  4. 选择打印类型（单选）
  5. 预览打印效果（模型渲染图 + 打印载体的模拟效果）
- **打印类型**:

| 打印类型 | 尺寸 | 价格 | 效果说明 |
|---------|------|------|---------|
| 相纸打印 | 6 寸 | ¥待定 | 高清照片纸，光面/哑面可选 |
| 精致卡片 | 8.5×5.4cm | ¥待定 | 铜版纸卡片，圆角切割 |
| 亚克力板 | 10×15cm | ¥待定 | 透明亚克力 + UV 彩印，含底座 |

- **逻辑规则**:
  - 可选项，默认为"不需要"
  - 打印类型单选
  - 选择后价格自动叠加
  - 打印内容：AI 生成模型的最佳视角渲染图（自动选取正面 45° 视角）
  - 后续可扩展更多打印类型

### 功能点 3：真人原照片打印（可选）

- **描述**: 将用户上传的原始全身照打印为实体
- **用户操作流程**:
  1. 在"可选周边"区域，展示"真人照片打印"卡片
  2. 开关式选择：不需要 / 需要
  3. 选择"需要"后展开打印类型选择
  4. 预览照片 + 打印效果
- **打印类型**: 与模型照片打印共用同一套打印类型和定价

| 打印类型 | 尺寸 | 价格 | 效果说明 |
|---------|------|------|---------|
| 相纸打印 | 6 寸 | ¥待定 | 高清照片纸，光面/哑面可选 |
| 精致卡片 | 8.5×5.4cm | ¥待定 | 铜版纸卡片，圆角切割 |
| 亚克力板 | 10×15cm | ¥待定 | 透明亚克力 + UV 彩印，含底座 |

- **逻辑规则**:
  - 可选项，默认为"不需要"
  - 与模型照片打印独立选择（两个可以都选，也可以只选一个）
  - 各自独立选择打印类型
  - 打印内容：用户上传的原始照片

### 功能点 4：包装盒选择（可选）

- **描述**: 选购与手办同材质的展示级包装盒
- **用户操作流程**:
  1. 在"可选周边"区域，展示"专属包装盒"卡片
  2. 开关式选择：不需要 / 需要
  3. 选择"需要"后展开颜色选择
  4. 8 种颜色以色块方式展示，点击选择
  5. 右侧/上方展示选中颜色的包装盒 3D 渲染效果图
- **包装盒规格**:

| 项目 | 说明 |
|------|------|
| 材质 | 与手办同材质 |
| 结构 | 一体式六面正方体，可展开/合拢 |
| 用途 | 展开取出手办 → 合拢后手办摆放在顶部 |
| 表面 | 暂无印刷内容，纯色 |
| 扩展 | 后续增加小卡扣（放置卡片/亚克力板） |
| 价格 | ¥待定 |

- **可选颜色（8 种基础色）**:

| 颜色名称 | 色值（参考） | 说明 |
|---------|------------|------|
| 经典白 | #FFFFFF | 简约百搭 |
| 雅致黑 | #2D2D2D | 高端质感 |
| 樱花粉 | #FFB7C5 | 甜美可爱 |
| 天空蓝 | #87CEEB | 清新活力 |
| 薄荷绿 | #98FF98 | 自然清爽 |
| 柠檬黄 | #FFF44F | 明亮活泼 |
| 薰衣草紫 | #E6E6FA | 优雅浪漫 |
| 珊瑚橙 | #FF7F50 | 温暖热情 |

- **逻辑规则**:
  - 可选项，默认为"不需要"
  - 颜色必须选择一种（选择"需要"后）
  - 包装盒尺寸根据手办尺寸自动匹配（S/M/L 对应不同包装盒尺寸）
  - 不同尺寸包装盒价格可能不同

### 功能点 5：套餐组合（可选）

- **描述**: 预设的优惠组合套餐，一键选购多个产品
- **用户操作流程**:
  1. 在定制页顶部或醒目位置展示"推荐套餐"区域
  2. 展示 2-3 个套餐卡片
  3. 点击套餐 → 自动勾选对应的产品组合
  4. 套餐价格低于单独购买总价，展示优惠金额
  5. 选择套餐后仍可微调单项（如更换包装盒颜色）
- **套餐定义**:

| 套餐名称 | 包含内容 | 套餐价 | 原价 | 优惠 |
|---------|---------|--------|------|------|
| 🌟 基础款 | 手办(S) | ¥0.01 | ¥0.01 | — |
| 💎 精致款 | 手办(M) + 包装盒 + 模型卡片打印 | ¥0.02 | ¥0.03 | 省¥0.01 |
| 👑 豪华款 | 手办(L) + 包装盒 + 模型亚克力板 + 真人相纸打印 | ¥0.03 | ¥0.04 | 省¥0.01 |

> ⚠️ 以上价格为 MVP 测试价，非真实售价。正式上线前由商务确定真实定价和优惠策略。

- **逻辑规则**:
  - 套餐与自由定制二选一（选套餐后自由定制项自动同步，反之亦然）
  - 选择套餐后微调单项，价格按套餐基础重新计算
  - 如果微调后的组合恰好等于另一个套餐，自动匹配该套餐价格
  - 套餐可由后台管理系统动态配置

### 功能点 6：实时价格计算与预览

- **描述**: 根据用户选择的所有定制项，实时计算并展示总价
- **页面展示**:

```
┌────────────────────────────────────┐
│  价格明细                          │
├────────────────────────────────────┤
│  Q 版手办（M 标准款）    ¥xxx.xx   │
│  模型照片打印（亚克力板）  ¥xx.xx   │
│  专属包装盒（樱花粉）     ¥xx.xx   │
├────────────────────────────────────┤
│  套餐优惠               -¥xx.xx   │
├────────────────────────────────────┤
│  合计                   ¥xxx.xx   │
│                                    │
│       [ 提交订单 ]                 │
└────────────────────────────────────┘
```

- **逻辑规则**:
  - 底部常驻价格栏：展示合计金额 + "提交订单"按钮
  - 每次选择变更，价格在 200ms 内更新（前端计算，无需请求后端）
  - 点击价格栏可展开价格明细弹窗
  - 套餐优惠以红色负数展示
  - 所有价格精确到分（¥xx.xx）
- **异常处理**:
  - 未选择必选项（手办尺寸）："提交订单"按钮置灰，提示"请先选择手办尺寸"
